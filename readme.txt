In the source_codes folder the code_PM_final_annotated.m file is the main source code, to be opened and run
with matlab. This file uses the function get_deme_info.m.

In the scenario_parameter_files folder there are seven .m files for the seven scenarios referred to in the paper.

_base: 		base scenario
_densdep: 	strong winter DD
_dispdist: 	dispersal distance
_no_dd: 	no winter DD
_no_dispersal: 	no dispersal
_var_migrants:	var survival migrants
_var_residents:	var survival residents
